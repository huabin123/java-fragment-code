# 06 实际应用案例

## Bean 拷贝工具

### 应用场景
- DTO 与 Entity 之间转换
- 对象深拷贝/浅拷贝
- 配置对象合并

### 实现示例
参考 `BeanCopyUtil.java`：
- 字段级别拷贝
- Getter/Setter 方法拷贝
- 类型匹配检查

## JSON 序列化框架

### 核心原理
1. 通过反射获取对象所有字段
2. 递归处理嵌套对象和集合
3. 处理泛型类型信息

### 关键技术点
- 泛型类型擦除处理
- 循环引用检测
- 自定义序列化器

## 依赖注入容器

### 实现步骤
1. 扫描类上的注解
2. 分析构造器和字段依赖
3. 创建对象实例并注入依赖

## ORM 框架

### 反射在 ORM 中的应用
- 实体类字段映射到数据库列
- 结果集到对象的转换
- 关联关系的懒加载

## 测试框架

### 单元测试中的反射
- 访问私有方法进行测试
- Mock 对象的字段注入
- 测试数据的自动生成
