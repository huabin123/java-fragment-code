## 快速入口

**BIO 服务无响应**：→ 第三章 3.1（未设 `SoTimeout`）、第二章 2.4（有界队列）

**服务重启报 "Address already in use"**：→ 第三章 3.1（`setReuseAddress(true)`）

**高并发下线程数爆炸**：→ 第二章 2.3（线程池模型）、2.4（线程数估算）

**何时从 BIO 切换到 NIO**：→ 第四章 4.3（BIO vs NIO vs Netty 选型）
