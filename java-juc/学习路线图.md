# Java并发编程(JUC)完整学习路线图

> 从基础到精通，系统掌握Java并发编程

---

## 📋 学习路线总览

```
第一阶段：理论基础（1周）
├── jmm - Java内存模型
│   ├── 可见性、原子性、有序性
│   ├── happens-before原则
│   ├── volatile深入
│   └── 内存屏障
└── 前置知识回顾
    ├── thread - 线程基础
    ├── threadpool - 线程池
    └── threadlocal - ThreadLocal

第二阶段：无锁编程（1周）
└── atomic - 原子类与CAS
    ├── CAS原理与ABA问题
    ├── AtomicInteger/Long/Boolean
    ├── AtomicReference系列
    └── LongAdder高性能原子类

第三阶段：显式锁（1周）
└── lock - Lock接口与实现
    ├── ReentrantLock
    ├── 公平锁vs非公平锁
    ├── Condition条件队列
    ├── ReadWriteLock读写锁
    └── StampedLock乐观锁

第四阶段：核心框架（1-2周）⭐
└── aqs - AbstractQueuedSynchronizer
    ├── AQS设计思想
    ├── 同步状态与CLH队列
    ├── 独占模式与共享模式
    └── 自定义同步器

第五阶段：同步工具（1周）
└── sync - 同步工具类
    ├── CountDownLatch
    ├── CyclicBarrier
    ├── Semaphore
    ├── Exchanger
    └── Phaser

第六阶段：并发容器（1周）
└── container - 并发容器
    ├── ConcurrentHashMap
    ├── CopyOnWriteArrayList
    ├── ConcurrentLinkedQueue
    ├── BlockingQueue系列
    └── ConcurrentSkipListMap

第七阶段：异步编程（1周）
└── async - 异步编程
    ├── Future与Callable
    ├── CompletableFuture
    ├── 异步编排与组合
    └── 异常处理与超时
```

## 💡 学习技巧

### 1. 循序渐进

```
不要跳跃学习，按照路线图顺序：
jmm → atomic → lock → aqs → sync → container → async
```

### 2. 动手实践

```
每个知识点都要：
1. 看文档理解原理
2. 运行demo观察现象
3. 修改代码实验
4. 完成project项目
```

### 3. 对比学习

```
对比相似的概念：
- volatile vs synchronized
- AtomicLong vs LongAdder
- ReentrantLock vs synchronized
- CountDownLatch vs CyclicBarrier
```

### 4. 画图理解

```
复杂概念用图表示：
- JMM的内存模型
- CAS的执行流程
- AQS的队列结构
- 线程状态转换
```

### 5. 源码阅读

```
阅读源码的顺序：
1. 先看接口定义
2. 再看核心方法
3. 最后看实现细节
```
