## 🚀 快速开始

### 1. 运行Demo代码

```bash
# 进入代码目录
cd src/main/java/com/fragment/jvm

# 编译代码
javac advanced/demo/*.java
javac memory/demo/*.java
javac gc/demo/*.java

# 运行示例
# JIT编译器演示
java -XX:+PrintCompilation advanced.demo.JITDemo

# 优化技术演示
java -XX:+DoEscapeAnalysis \
     -XX:+PrintEliminateAllocations \
     -XX:+UnlockDiagnosticVMOptions \
     advanced.demo.OptimizationDemo

# 内存结构演示
java -Xms256m -Xmx256m \
     -XX:+PrintGCDetails \
     memory.demo.MemoryStructureDemo

# GC收集器演示
java -XX:+UseG1GC \
     -XX:+PrintGCDetails \
     gc.demo.GCCollectorDemo
```

### 2. 运行项目代码

```bash
# JVM性能分析器
java advanced.project.JVMProfiler

# 内存监控工具
java memory.project.MemoryMonitor

# GC分析工具
java gc.project.GCAnalyzer
```
